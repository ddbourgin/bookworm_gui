About
The current browser allows you to search for trends within the transcripts of the nationally syndicated radio show This American Life. It was built using Bookworm with transcripts collected from the This American Life radio archives.

Operation
The browser supports displaying multiple trends on the same plot. To add a new trend line to the current plot, click the green "+" symbol to the right of the search box. For now, the maximum query length is 3 words.

You can adjust the display parameters for the plot by clicking on the gear icon at the upper right hand corner of the page.

  • The Time field controls the variable displayed on the horizontal axis of the chart -- you can plot the results as a function of the episode number, the episode year, or the within-episode timestamp in minutes.
  • The Metric field allows you to select whether the results are displayed in terms of the raw number of occurences of the search query within the corpus (word count; unnormalized) or in terms of the ratio of the number of query occurrences to the total number of terms in the corpus (% of words; normalized).
  • The Case field determines whether or not the search is case-sensitive.
  • The Smoothing slider allows you to choose the number of adjacent datapoints to average over when displaying the search results. Larger values of this parameter result in smoother trendlines, but run the risk of obscuring local variation in the data.

Notes
Be aware that the viewer is still a bit buggy! In particular, it may sometimes require re-running a particular query in order to ensure that a recent change in search parameters is fully reflected in the results. It has also been observed that punctuation (in particular, apostrophes when used in contractions) cause the viewer to break :-(
